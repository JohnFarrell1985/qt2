# 机器学习模块 (src/ml/)

## 概述

使用 LightGBM 对因子组合建模，预测个股未来收益率，自动迭代优化因子组合。

## 模块结构

| 文件 | 职责 |
|------|------|
| `dataset.py` | 构建训练数据集 (因子矩阵 + 标签) |
| `lgb_model.py` | LightGBM 模型封装 (训练/预测/保存) |
| `feature_selection.py` | IC/相关性因子筛选 |
| `model_evaluation.py` | 模型评估 (IC/ICIR/分组收益) |
| `strategy_builder.py` | 预测结果→交易信号 |
| `auto_iterate.py` | 自动迭代优化引擎 |

## LightGBM 模型

### 默认参数
```python
{
    "objective": "regression",
    "metric": "mse",
    "boosting_type": "gbdt",
    "num_leaves": 63,
    "learning_rate": 0.05,
    "n_estimators": 1000,
    "subsample": 0.8,
    "colsample_bytree": 0.8,
    "importance_type": "gain",
}
```

### Early Stopping
LightGBM >= 4.0 要求通过 callback 传递:
```python
callbacks=[
    lgb.early_stopping(stopping_rounds=50),
    lgb.log_evaluation(period=100),
]
```

### 模型序列化
使用 `joblib.dump/load` 保存完整 sklearn 包装器 (保留 `best_iteration_` 等属性)。

## 数据集构建

```
因子数据 (factor_values 表)
    │
    ▼  get_factor_values() → MultiIndex(trade_date, code) DataFrame
    │
    ▼  _calc_forward_returns() → 未来N日收益率标签
    │
    ▼  内连接 → (X: 因子矩阵, y: 标签)
    │
    ▼  时序分割 (60%训练 / 20%验证 / 20%测试)
```

## 自动迭代优化 (AutoIterateEngine)

### 工作流程

```
1. IC/IR 初筛 → top_n 因子
        │
2. LightGBM 训练 + 验证
        │
3. 综合评分 = IC×0.3 + ICIR×0.4 + 多空收益×0.3
        │
4. 因子进化:
   - 淘汰底部 20% 低重要性因子
   - 随机引入新候选因子
   - 每3轮探索3个新因子
   - 每5轮随机调参 (num_leaves, lr, n_estimators)
        │
5. 收敛判定:
   - 达到 target_sharpe → 停止
   - 连续 patience 轮无改善 → 停止
   - 达到 max_iterations → 停止
        │
6. 输出: 最佳因子组合 + 权重 + 模型
```

### 统计分析

- **因子频率**: 统计各因子在所有迭代中出现次数
- **收敛曲线**: 每轮得分 + 累计最佳得分
- **因子重要性**: LightGBM `feature_importances_` (gain)
- **DB 记录**: 每轮结果存 `ml_model_log` 表
