# 交易模块 (src/trading/)

## 概述

对接迅投 QMT 的模拟盘和实盘交易，包含完整的风控体系。

## 模块结构

| 文件 | 职责 |
|------|------|
| `qmt_trader.py` | QMT 交易客户端封装 |
| `order_manager.py` | 委托管理 (下单/撤单/记录) |
| `position_manager.py` | 持仓管理 (查询/快照) |
| `risk_control.py` | 风控模块 (仓位/止损/止盈) |
| `paper_trading.py` | 模拟盘引擎 |
| `live_trading.py` | 实盘引擎 (继承模拟盘 + 安全确认) |
| `trade_log.py` | 每日绩效记录 |

## QMT 交易 API

参照 [xttrader 官方文档](https://dict.thinktrader.net/nativeApi/xttrader.html)。

```python
trader = QMTTrader(account_id="your_account", mini_qmt_path="C:/MiniQMT/userdata_mini")
trader.connect()

# 下单 (order_type → xtconstant.STOCK_BUY / STOCK_SELL)
order_id = trader.order_stock("600519.SH", "buy", 100, price_type="LATEST_PRICE")

# 查询
asset = trader.query_asset()       # {total_asset, cash, market_value, frozen_cash}
positions = trader.query_positions() # [{code, volume, can_use_volume, ...}]
orders = trader.query_orders()       # [{order_id, code, direction, ...}]
```

## 风控体系

| 风控规则 | 默认值 | 说明 |
|---------|--------|------|
| 单股仓位上限 | 30% | 单只股票占总资产比例 |
| 总仓位上限 | 80% | 总持仓占总资产比例 |
| 止损线 | -8% | 浮亏达标则强制卖出 |
| 止盈线 | +20% | 浮盈达标则提示卖出 |
| 日亏损限额 | -5% | 日内亏损达标停止交易 |

## 交易流程

### 模拟盘
```
ML信号 → execute_signals() → 风控检查 → QMT下单 → 记录DB
                                                        │
定时任务 ─── check_risk_and_stop() → 止损/止盈卖出      │
                                                        │
收盘 ─────── daily_close() → 持仓快照 ─────────────────┘
```

### 实盘
在模拟盘基础上增加:
- **二次确认**: 必须调用 `confirm_live_mode()` 才能执行
- **account_type = "live"** 区分日志和快照
