# 策略管理模块 (src/strategy/)

## 概述

策略池化管理、标的池化、宏观环境状态映射，实现多策略自动编排。

## 模块结构

| 文件 | 职责 |
|------|------|
| `strategy_pool.py` | 策略 CRUD、排名 |
| `instrument_pool.py` | 标的池管理 (静态/动态) |
| `macro_env.py` | 宏观环境状态管理 |
| `orchestrator.py` | 策略编排器 (宏观→策略→标的) |

## 策略池

一个策略 = 一组因子 + LGB模型参数 + 训练好的模型 + 回测指标。

### 策略生命周期

```
创建 (create_strategy)
  │
  ▼ 自动迭代训练
  │
  ▼ 回测验证 → 更新指标 (update_backtest_metrics)
  │
  ▼ 入池 (status=active)
  │
  ├── 绑定标的池 + 宏观环境
  │
  ├── 暂停 (status=paused)
  │
  └── 归档 (status=archived)
```

### 策略属性

| 字段 | 说明 |
|------|------|
| factor_names | 因子列表 (JSON) |
| factor_weights | 因子权重 (JSON, 来自 LGB importance) |
| model_params | LGB 模型参数 (JSON) |
| model_path | 训练好的 .pkl 文件路径 |
| backtest_sharpe | 回测夏普比率 |
| backtest_annual_return | 回测年化收益 |
| backtest_max_drawdown | 回测最大回撤 |
| applicable_macro | 适用的宏观状态 (逗号分隔) |

## 标的池

### 静态池
手动指定股票列表:
```python
pool_mgr.create_pool("白马股", codes=["000001", "600519", ...])
```

### 动态池
根据规则自动筛选:
```python
pool_mgr.create_pool("大市值", filter_rules={"min_market_cap": 500})
pool_mgr.refresh_dynamic_pool("大市值")  # 从 stocks 表查询
```

### 内置池
- 沪深300, 中证500, 创业板, 科创板, 大市值, 中小市值

## 宏观环境状态

配置文件: `macro_env.json` (项目根目录)

### 6 种宏观状态

| 状态 | 仓位乘数 | 推荐策略 |
|------|----------|---------|
| bull_strong (强势牛市) | 1.2 | momentum_aggressive, growth_factor |
| bull_moderate (温和牛市) | 1.0 | balanced_growth, quality_factor |
| range_bound (震荡市) | 0.8 | mean_reversion, low_volatility |
| bear_moderate (温和熊市) | 0.5 | defensive, value_factor |
| bear_severe (深度熊市) | 0.3 | cash_heavy, defensive |
| recovery (底部复苏) | 0.9 | value_factor, contrarian |

### 状态切换
```python
macro_env.set_current_state("bull_moderate", determined_by="manual")
```
切换后自动记录到 `macro_state_log` 表。

## 策略编排器

```
当前宏观状态 (range_bound)
        │
        ▼ 过滤: 排除 avoid_strategies
        │
        ▼ 匹配: strategy_allocation 表
        │
        ▼ 调整: weight × position_multiplier
        │
        ▼ 输出: 执行计划 (策略+标的池+权重)
```

API: `GET /strategy/plan` 一键获取当前执行计划。
