# qt-quant 总体设计 (v4)

## 1. 系统定位

面向 **A 股 T+1** 的精简量化流水线：

1. **数据采集落盘** — QMT 为主、多源 fallback，写入 PostgreSQL
2. **程序选股** — 多周期均线规则初筛 + 综合打分，输出候选清单
3. **回测 / 交易** — 基于清单的策略回测；QMT 模拟盘 / 实盘执行

## 2. 架构总览

```
              ▼                           ▼

```

## 3. 核心流程

### 3.1 每日选股

```
16:30  scheduler 增量同步 K 线
  ↓
python -m src.selection screen --date YYYY-MM-DD
  ↓
reports/candidates_{strategy}_YYYYMMDD.json
```

### 3.2 回测与交易

```
scripts/run_backtest.py --picks-file reports/candidates_*.json
python -m src.trading execute --picks reports/candidates_*.json --mode paper
```
