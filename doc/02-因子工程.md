# 因子工程模块 (src/factor/)

## 概述

因子工程处理链: 原始数据 → 因子计算 → 预处理 → 单因子检验 → 因子池。

## 模块结构

| 文件 | 职责 |
|------|------|
| `factor_pool.py` | 因子元信息管理、查询 |
| `factor_calc.py` | 自定义技术/动量因子计算 |
| `factor_preprocess.py` | 去极值/标准化/中性化 |
| `factor_analysis.py` | IC/IR 分析、分组回测 |

## 因子预处理流水线

```
原始因子值
    │
    ▼ winsorize (MAD 去极值, ±3σ)
    │
    ▼ standardize (Z-score 标准化)
    │
    ▼ neutralize (行业+市值中性化, OLS 回归残差)
    │
    ▼ 标准化后的因子值
```

### 去极值 (MAD)
使用中位数绝对偏差 (MAD) 替代标准差，对异常值更稳健:
```
MAD_e = 1.4826 × MAD
上界 = median + 3 × MAD_e
下界 = median - 3 × MAD_e
```

### 中性化
通过 OLS 回归消除行业和市值效应:
```
factor = β₀ + β₁×industry_dummies + β₂×ln(market_cap) + ε
中性化因子 = ε (残差)
```

## 单因子检验

| 指标 | 阈值 | 说明 |
|------|------|------|
| IC 均值 | > 0.02 | Spearman Rank IC |
| ICIR | > 0.3 | IC均值/IC标准差 |
| IC 正向率 | > 55% | IC > 0 的占比 |
| 分组单调性 | G1 < G5 | 多空收益显著 |
