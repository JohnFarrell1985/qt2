# 回测引擎 (src/backtest/)

## 概述

支持日线和分钟线两种粒度的策略回测，包含完整的费用计算和绩效统计。

## 模块结构

| 文件 | 职责 |
|------|------|
| `engine.py` | 单笔交易计算、多笔汇总 |
| `engine_minute.py` | 分钟线回测引擎 |
| `fees.py` | A股 + 港股通费用计算 |
| `performance.py` | 绩效统计 (夏普/回撤/Calmar等) |
| `data_loader.py` | 从 PostgreSQL 加载行情数据 |
| `strategy_runner.py` | 策略运行器 |
| `stock_picker.py` | 选股器 (Mock/Random) |
| `cli.py` | 命令行回测入口 |

## 绩效指标

| 指标 | 公式 |
|------|------|
| 年化收益率 | (终值/初值)^(365/天数) - 1 |
| 最大回撤 | max((peak - trough) / peak) |
| 夏普比率 | (年化收益 - 无风险利率) / 年化波动率 |
| Sortino比率 | (年化收益 - rf) / 下行波动率 |
| Calmar比率 | 年化收益 / 最大回撤 |
| 胜率 | 盈利交易 / 总交易 |
| 盈亏比 | 平均盈利 / 平均亏损 |

## 分钟线回测

```python
from src.backtest.engine_minute import MinuteBacktestEngine

def my_strategy(bars, position, cash):
    if len(bars) >= 20 and bars[-1]["close"] > bars[-20]["close"]:
        return "buy"
    if position > 0 and bars[-1]["close"] < bars[-5]["close"]:
        return "sell"
    return None

engine = MinuteBacktestEngine(my_strategy, period="5m")
result = engine.run("000001", date(2025, 1, 1), date(2025, 6, 30))
```
