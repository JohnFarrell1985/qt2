# 选股模块 (src/selection/)

## 概述

**启动突破 / 熊市反弹** 两套 preset，手动切换。程序初筛 → **综合打分排序** → 输出 JSON/CSV。

输出 `reports/candidates_{strategy}_{date}.json`（按 `composite_score` 降序，含 tier A/B/C）。

## 策略 preset

| ID | 文件 | 场景 |
|----|------|------|
| `bull_launch` | `config/strategies/bull_launch.json` | 牛市启动突破：5/10/20/50 发散，贴 MA5 ±3% |
| `bear_rebound` | `config/strategies/bear_rebound.json` | 熊市反弹：20~60 多头上行，贴 MA20 ±5% |

全局开关：`config/app.json` → `selection.active_strategy`。CLI：`--strategy bear_rebound`。环境变量：`SELECTION_STRATEGY`。

## 模块结构

| 文件 | 职责 |
|------|------|
| `ma_screener.py` | 硬筛 + 综合打分 + tier 分层 |
| `workflow.py` | screen 编排与 JSON/CSV 输出 |
| `__main__.py` | CLI：`screen` |

## 硬筛规则（须同时满足）

> **「日」= 交易日**：数据来自 `stock_daily` 连续 K 线。

配置见当前策略 JSON 的 `ma_filter` 段（如 `config/strategies/bull_launch.json`）：

| 参数 | 说明 | 默认 |
|------|------|------|
| 向上发散 | 筛选日多头排列 + 抬高 + 间距扩大 | 开 |
| `prior_surge_lookback_days` / `prior_surge_min_pct` | 前 N 日有大涨 | 5 / 5% |
| `max_gain_lookback_days` / `max_gain_total_pct` | 近 N 日累计涨幅上限 | 10 / 30% |
| `require_volume_pullback` | 当日相对前一日缩量 | true |
| `require_ma5_proximity` | **硬性**：收盘不得偏离锚点均线超过 `ma5_proximity_pct` | **true** |
| `anchor_ma_period` | 锚点均线周期（牛市 5 / 熊市 20） | 5 |
| `ma5_proximity_pct` | 相对锚点均线允许偏离 (%)，上下对称 | **3.0** (bull) / **5.0** (bear) |
| `require_low_above_ma5` | 可选：最低价不破锚点均线 | false |

## 综合打分 `rank`（策略 JSON 内）

| 维度 | 权重 | 说明 |
|------|------|------|
| 贴锚点均线 | 30% | 收盘距锚点均线越近越高分 |
| 缩量 | 25% | `vol_shrink_ratio` 越小越高分 |
| 10 日涨幅 | 20% | 8～18% 最佳，接近上限降分 |
| 大涨新近度 | 15% | 大涨越靠近筛选日越高分 |
| 流动性 | 10% | 20 日均成交额 |

分层：`tier A` ≥70，`B` ≥55，其余 `C`。候选按 `composite_score` **降序**输出。

快照字段：`composite_score`、`tier`、`days_since_surge`、`low_ma5_dist_pct` 等。

## 推荐工作流

```bash
# 盘后 sync 后初筛 + 审计 + 排序分布
uv run python scripts/run_screen_audit.py --date 2026-03-11
uv run python scripts/run_screen_audit.py --date 2026-03-11 --strategy bear_rebound

# CLI 直接输出 JSON/CSV
uv run python -m src.selection screen --date 2026-03-11 --csv
```

## CLI

```bash
uv run python -m src.selection screen --date 2026-03-11 --strategy bull_launch
uv run python scripts/run_screen_audit.py --date 2026-03-11 --rank-dist --no-screen
```
